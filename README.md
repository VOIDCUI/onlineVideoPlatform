# onlineVideoPlatform

## 网课视频平台 - 毕设设计 - VOIDCUi

## 使用技术: Vue+NestJs +Element+MongoDB+NuxtJs

### 项目介绍: 该项目前端页面主要基于 Vue，页面分为后台管理端和网页端，管理端有首页、课程管理页面、课时管理页面和用户管理界面。而主站网页端页面由视频首页、热门课程页面和视频播放页面组成，采用 NestJs 作为我们后端，配合 MongoDB 来完成 api 接口的开发，用户可以按照课时上传教学视频，视频主站页面使用了 NuxtJs 来进行服务器渲染，提高网站 SEO。
